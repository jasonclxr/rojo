## Getting Started
To build the place from scratch, use:

```bash
rojo build -o "rojo.rbxlx"
```

Next, open `rojo.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
